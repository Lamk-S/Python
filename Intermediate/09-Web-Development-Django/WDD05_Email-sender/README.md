Un proyecto enfocado en la integración de un backend robusto con **Django** y la programación del lado del cliente con **JavaScript**. Esta aplicación permite listar empleados desde una base de datos MySQL, visualizar los datos mediante *DataTables* y gestionar el envío de correos electrónicos interactuando dinámicamente con el DOM.

## 🎯 Objetivos del Proyecto
* Implementar arquitectura MVT (Model-View-Template) con Django.
* Integrar librerías de JavaScript de terceros (DataTables, SweetAlert) para mejorar la Experiencia de Usuario (UX).
* Manejo seguro de credenciales y variables de entorno (`.env`).
* Conexión y gestión de base de datos relacional (MySQL).

## ⚙️ Requisitos Previos

Asegúrate de tener instalado en tu sistema:
* [Python 3.10+](https://www.python.org/downloads/)
* [MySQL Server](https://dev.mysql.com/downloads/installer/)
* `pip` (Gestor de paquetes de Python)
